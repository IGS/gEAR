Dataset Manager

This area is where you can manage your gEAR layout profiles and datasets. You can create new profiles, remove old ones, and set a profile to be used for you gEAR searches. Datasets can also be added, removed, and rearranged within profiles to better fit your needs. In this area, you can also change a dataset's access level, remove it from gEAR, and share it with others.

Profile Management

This area manages the gEAR layout profiles. A layout profile is used by gEAR to display gene expression results. It contains datasets, and the order of those datasets, that display the expression values of the searched gene(s). A profile can be set as primary, created, shared, or deleted. Multiple profiles can be created, but only one profile can be set as primary and used by gEAR to display search results at one time.


The 'Site Default' profile is available to all gEAR users and guests. It cannot be modified, or deleted. To create a profile that is more tailored to your needs, first create a gEAR account and log in (Learn how to create a gEAR account).


Set as Primary

To Set as primary, click . This sets the profile for use by gEAR. When using gEAR the profile set as primary will be used to display search results.


Add or Delete Profile

To add new a profile, click . Next, enter a profile name and click to save the profile. Now, you can add datasets to the profile by searching your datasets and public datasets.

To Delete a profile, click . This will not delete any of the datasets that have been added to it.


Share a Profile

To share a profile, click . Next, use the URL link provided to share the profile with someone.

Profiles containing private datasets cannot be shared, unless you own them. Please note that private datasets will be shared.



Views

List view is the first view that displays in the Dataset Manager. In this view profiles and datasets can be managed and searched.

Arrangement view is where datasets are rearranged within a profile and resized.

Search Datasets

This area searches gEAR datasets. You can search for your datasets and public datasets in the corresponding fields like the field below:


Your datasets searches datasets you have uploaded to gEAR - public and private. Once you upload a dataset, search for it here, and add it to a profile.

Public datasets searches datasets labelled as 'public' by the dataset owner. These datasets are viewable to all gEAR users. Once you find a dataset, you can add it to a profile.


To list datasets that are shared with you, click .



Filter Datasets

Filter datasets to limit the datasets displayed. Select one or more organism(s) to show only those organisms. Datasets can be filtered by organism and graph type.



Edit Datasets

Information about each dataset can be modified. The following information can be editted: schematic image, title, description, PubMed ID, and GEO ID. Only the dataset owner can access this function.
