Profile


In this area users can:

  • Edit, create, and delete profiles.
  • Change the working profile for exploring gEAR.

Having the ability to create a new profile means the user can organize the datasets in a flow that best fits his or her needs. Once a new profile is added, datasets can be added to the new profile in the List View. Then the datasets can be rearranged in the Arrangement view.
