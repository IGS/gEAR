View types

The dataset manager has 2 views, which can be changed via the List - Arrangement button group in the column on the left:

Each view provides different functions to the user. List View is where the user can change the access level, share, view, or even, delete a particular dataset. In the Arrangement view, the datasets within the selected Profile can be rearranged. This customizes the profile to better the user's needs. For further details on each view, continue below.

  • List view
  • Arrangement view

List View

    • Under construction

    • Add the dataset to the current workspace Profile.

    • Change to Private or Public. Only the owner of the dataset can change the acccess level. Private access level will make the dataset viewable to only the owner. If the owner has shared the dataset with other users, it will still be viewable to them. Changing who the dataset is shared with can be managed with the Share button below.

    • Remove dataset from the gEAR. Only the owner of the dataset can remove the dataset.

    • Use the provided URL link to share the dataset with others.


Arrangement View

Rearrange the datasets into a customized layout. This layout determines how the datasets appear when searching the gEAR when using the selected Profile.

Simply click a dataset within the layout grid, then drag and drop it. When finished, be sure to click Save to save the new layout.
