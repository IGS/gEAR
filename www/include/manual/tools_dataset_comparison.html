Dataset Comparison

This analysis tool allows you to delve into datasets by comparing:

  • Condition A vs Condition B in Dataset X
  • Condition A of Dataset X vs Condition B of Dataset Y

These comparisons are drawn in an interactive X-Y scatterplot and allows you to resolve specific genes.


Getting Started

To open the dataset comparison tool click Compare in a dataset's drop-down menu.


This will bring you to the dataset comparison page.


Getting A Comparison

To plot a dataset comparison, use the drop-down menus on the left to select a dataset and condition for the X axis and Y axis. Then click Plot.

This will produce an x-y scatterplot like the one below. On the lower left, additional settings allow you to display values as raw, Log2, and Log10 and filter data by fold change and standard deviation.


Plot Controls

Hover over the plot and you'll see buttons in the upper right, which control actions like zooming and selection of points.

Control Icons

The Box and Lasso selection tools allow you to click and drag a shape around genes of interest.

Selecting Genes

After you select your desired genes, a list of them all will be displayed. This gene list can be saved and used for other queries throughout gEAR.
