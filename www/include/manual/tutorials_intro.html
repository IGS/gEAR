gEAR Tutorials

These tutorials will help you get accustomed to the gEAR. As we add continue to develop and enhance the gEAR, we will also add new tutorials.

  • Create Layout Profile & Add Datasets to It

    Demonstrates how to create a new layout profile. This also shows how to add datasets to the new profile and make the profile usable in gene searches by using 'set as primary'.
