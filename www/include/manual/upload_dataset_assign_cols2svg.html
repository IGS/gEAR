SVG - Assigning Columns

This section falls under the "Organize your data" step of the dataset uploading process, and only applies to SVG-based uploads.


Upload SVG file

By this point you will have excluded text columns and renamed expression data columns (if necessary).

When prompted, upload your SVG file and you should see the column headers and SVG similar to the example below.


Select Column Header

With the SVG uploaded, you now need to assign expression data columns to the SVG path elements. To do this, first click a column header to select it.


Assign Column Header to SVG Element

Next, hover your mouse over the SVG path element that the column will be assigned to.


Click the SVG path element to assign the expression column to it. Repeat the process until you have assigned all of your expression columns to the SVG. When finished, click Continue to go to the next step.

To deselect, or unassign an SVG element, simply double-click that element.

Note: Any "greyed-out" standard deviation and p-value columns will automatically be included in the upload.

Once finished assigning expression columns to the SVG, you are ready to complete the next step:
