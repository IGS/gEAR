Bar graph format

To upload a dataset as a bar graph, gEAR requires only a tab-delimited text data file. This file holds your expression data, and should have a 'Gene symbol' column and then expression value columns, with headers on the first row.

Data file

This file must have a column labeled 'Gene symbol' containing your genes of interest. The following data columns contain the expression values. These columns must be labeled using the following convention: Group--Label. A Group can be considered your experimental groups, while a label can be considered a replicate. To use multiple words within a group or label, use an underscore ('_') instead of a space.

To include p-values and standard deviations, you use the exact name for the p-value and stand deviation columns as used for their the corresponding expression column and append '_pval' or '_sd'. For example, the p-value and stand deviation columns for column 'Untreated--P_0' would be 'Untreated--P_0_pval' and 'Untreated--P_0_sd', respectively. See below for an example.
