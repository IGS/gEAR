Step 1: Select Display Type

Datasets can be uploaded as four display types:

  1. SVG - Displays your data as a schematic/self-colorizing cartoon. In addition to a data file containing expression values, this display type requires a vector-based schematic file.
  2. Bar graph - Displays your data as a standard bar graph. This display type requires only a data file containing expression values.
  3. Line graph - Displays your data as a line graph. This display type requires only a data file containing expression values.
  4. Violin plot - Displays your data as a violin plot - typically for single cell datasets. This display type requires only a data file containing expression values.
