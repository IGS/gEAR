Organize Your Data

This step allows you to:

  • Exclude unwanted data columns
  • Rename and rearrange column headers
  • Assign expression columns to SVG path elements (SVG only)
  • Create subplots & preview graph (Bar & Line graphs only)

Violin plots do not have this functionality.


Select Unwanted Columns

The dataset uploader only requires a gene symbol column and numeric expression data columns. All other columns ought to removed prior to upload, but this step allows you to exclude these columns within the browser.

Columns to exclude include those that are:

  • Unrelated to the data you wish to visualize
  • Text, for example GO terms, gene biotypes, source of gene symbol

Below is an example of how the spreadsheet previewer will automatically pre-select columns that ought to be excluded.


To exclude a column from upload, click the column header and it will "grey-out". When finished, click Continue to move to the next step. This will hide the selected columns and data rows, leaving only the column headers. In the example below, additional columns have been excluded.


Rename & Rearrange Columns

With unnecessary columns removed, now the remaining columns can be renamed and rearranged.

Rename

To rename a column, simply click, enter the new name, and save.

Rearrange

To rearrange a column, simply click and drag the column.

SVG column headers cannot be rearranged.


The next step depends on the visualization type you have choosen for you dataset.
