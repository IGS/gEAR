Bar & Line Graphs - Creating Subplots

This section falls under the "Organize your data" step of the dataset uploading process, and only applies to bar graph and line graph uploads.


Select Layout

Here you can choose to keep the graph as 1 plot, or subdivide the graph into 2 or 3 subplots.

Bar and line graphs can be subdivided into 2 or 3 plots. Bar graphs will horizontally subdivide the graph space, so the subplots will aline in a row. Line graphs vertically subdivide the graph space, so the subplots will "stack" on top of each other.

In this example, we will choose "2 plots".


Once we select a layout, fields for a plot title (optional) and column headers will appear for each subplot.


Select & Move Column Headers

Now we can move the column headers into the subplots with a simple "click and drag". To select multiple column headers, hold ctrl then click on the column headers. Next, drag the group into the subplot.

Note: Any column headers not moved into a subplot will be excluded from the upload.


Preview Graph

After the column headers are placed in each subplot, click Get graph preview to preview the graph.

The graph preview is colored to correspond with each column header. All standard deviation ('_sd') and p-value ('_pval') columns will be grayed out. The graph is generated using the first row of expression data from the data file. In the example, Pbsn is the graphed gene.


If you do not like how the graph is arranged, you can rearrange the column headers, even change the number of subplots.

When finished rearranging the graph, click Get graph preview to get an updated preview.
