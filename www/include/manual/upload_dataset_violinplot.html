Violin plot format

To upload a dataset as a violin plot, gEAR requires only a data file. This file holds your expression data, and should have a 'Gene symbol' column and then expression value columns, with headers on the first row.

Data file

This file must have a column labeled 'Gene symbol' containing your genes of interest. The following data columns contain the expression values. These columns must be labeled using the following convention: Group--Label. A Group can be considered your experimental groups, while a label can be considered a replicate or individual cell. To use multiple words within a group or label, use an underscore ('_') instead of a space. See below for an example.
