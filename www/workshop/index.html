EARssentials 2023

Workshop schedule

Part 1 - Introduction

This will be a walk-through of the basic features of the portal, giving the background needed for the more tool-specific sessions. Topics include account creation, basic gene expression searches, data sharing, site navigation and overall feature discussion. We will also overview some of the new features of gEAR and point users to online resources where they can learn more about them.

View workshop slides

Part 2 - Explore and Analyze Basics: Creating gene carts and dataset comparisons

Learn how to create collections of genes (carts) which can be used with many tools on the portal. Then, see how the dataset comparison tool is used to visualize differences in expression between two conditions of the same dataset, with statistical tests.

View workshop slides

Part 3 - Analyze scRNA-seq data: The single-cell workbench

Learn how to use the workbench without any programming experience, walking through all the steps of a scRNA pipeline using a point and click interface.

View workshop slides

Part 4 (time permitting) - Data upload

Users can upload their own datasets to the gEAR and view/analyze them privately, share with others specifically, or make them public entirely. Here we cover the required input formats and steps for uploading.

For this workshop, we'll need a few demonstration files if you'd like to follow along. Just download these and remember where you've saved them. We'll cover what they are during the workshop.

  • Expression data (demo_bulk.tar.gz)
  • Metadata sheet (demo_metadata.xlsx)
